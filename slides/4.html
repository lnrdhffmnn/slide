Pontos fortes

  • Versatilidade

    É possível desenvolver um aplicativo completo em JavaScript, do front-end ao back-end

  • Interfaces ricas

    O JS oferece várias interfaces e componentes que melhoram a interatividade com o usuário.

  • Baixa curva de aprendizado

    O JS é uma ótima escolha para iniciantes que desejam criar aplicativos web.

  • Popularidade

    Como todos os navegadores modernos suportam JavaScript, ele está presente praticamente em todos os lugares.

  • Documentação completa

    JavaScript é bem documentado, facilitando a busca por soluções para erros e bugs.

Pontos fracos

  • Compatibilidade

    Existem casos em que o JavaScript é interpretado de forma diferente pelos navegadores, especialmente ao usar o DOM.

  • Controle do codigo

    JavaScript público pode ser controlado e reutilizado facilmente por outros desenvolvedores.

  • Seguraçao client-side

    Devido ao fato de que o código JavaScript é visível para o usuário, outros podem utilizá-lo com propósitos maliciosos, comprometendo a segurança dos dados do site.
